Jogo de Cartas

Regras do Jogo:

  • O jogo é jogado entre dois jogadores.
  • Cada jogador recebe X cartas.
  • O jogador que tiver a carta mais alta vence a rodada.
  • O jogo continua até que um jogador vença X rodadas.

Área dos jogadores

Jogador 1

  • Carta 1: Ás de Copas (Valor: 11)
  • Carta 2: Dois de Paus (Valor: 2)
  • Carta 3: Três de Ouros (Valor: 3)

Jogador 2

  • Carta 1: Rei de Espadas (Valor: 10)
  • Carta 2: Dama de Copas (Valor: 8)
  • Carta 3: Valete de Ouros (Valor: 6)
